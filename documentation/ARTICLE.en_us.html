Akt — a powerful typeface for modern user needs

Unified glyphs width across all weights (Thin → Black) helps keep line lengths and UI element widths predictable when adjusting text weight. It also aids dark-theme tuning: step one weight lighter to balance contrast.

Capitals are vertically centered within the line height for harmonious alignment with icons. Terminals are horizontally calibrated for visual consistency.

Over 1,200 glyphs with support for 400+ languages across Extended Latin and Cyrillic. There are 7 font weights available (Thin → Bold ready; ExtraBold and Black are in progress). Designed by Dima Grenev.

Key points

  • Unified glyph width across the family
  • Centered capitals and calibrated terminals
  • Extensive Latin and Cyrillic coverage
  • Seven weights for flexible UI and text use
